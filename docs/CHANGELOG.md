# 更新日志

## [1.0.2] - 2024-01-29

### 🔨 优化

- 📝 表单优化
  - 修改所有表单中的备注字段为非必填
  - 优化收入和支出表单的用户体验
  - 统一表单字段验证逻辑

## [1.0.1] - 2024-01-29

### 🐛 修复

- 🔄 请求优化

  - 添加 `AbortController` 管理请求生命周期
  - 优化组件卸载时的请求清理机制

- ✅ 完整的基础组件库

- 🎨 主题系统

- 📝 操作日志记录
